# support-vector-clustering
Реализация алгоритма кластеризации на основе метода опорных векторов (SVC) с нуля. Исследование влияния гиперпараметров на качество кластеризации сложных данных.
## Цель проекта
Показать применение SVC для выделения кластеров произвольной формы, когда стандартные методы (например, k-means) не справляются.  

## Что сделано
- Реализован алгоритм **SVC** с RBF-ядром.  
- Исследовано влияние гиперпараметров **q** и **p**.  
- Проанализировано количество **опорных векторов (SV)** и **пограничных векторов (BSV)**.  
- Проведены эксперименты на синтетических наборах данных ("кольца", "луны", "облака").  
- Выполнена визуализация кластеров и границ.
